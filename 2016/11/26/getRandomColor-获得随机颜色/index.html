getRandomColor()-获得随机颜色

此前在FCC前端开发项目中有一个弹幕APP中的JS代码有这样一段代码：

var getRandomColor = function() {
        return '#' + (function(h) {
            return new Array(7 - h.length).join("0") + h
        })((Math.random() * 0x1000000 << 0).toString(16))
    }

这段代码的作用是声明一个函数getRandomColor用于获取随机颜色，仔细阅读这个函数可以得到以下知识点：

Math.random()

定义

概述（MDN文档）
Math.random() 函数返回 [0-1) 的浮点值伪随机数（大于等于0，小于1）。

此函数的随机数生成器和java一样以当前时间为随机数种子。

用法举例

  1. 返回一个大于等于0，小于1的伪随机数

    function getRandom(){
    return Math.random();
}

  2. 返回一个介于min和max之间的随机数

    function getRandomRange(min, max){
    return Math.random()*(max - min) + min;
}

  3. 返回一个介于min和max之间的整型随机数

    16进制数

    0x在JS中表示16进制数前缀，0x1000000

    位操作

    new Array(num).join(“0”) 前置num-1个0
